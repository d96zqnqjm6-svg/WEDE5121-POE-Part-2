Aurora Hands of Hope – Website Project (Part 2)

Project Overview
This website is for Aurora Hands of Hope (AHOH), a non‑profit organisation based in Durban, South Africa. The site provides information about AHOH’s programmes, allows users to volunteer or sponsor, and includes a contact form.

Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (mobile menu toggle)
- Google Fonts
